# 🧠 Personalized Mental Health Check-In Bot

A web-based chatbot that interacts with users, detects their emotional tone using sentiment analysis, and provides personalized mental wellness suggestions. Built with web development tools and powered by a pre-trained sentiment analysis API — no machine learning model training required.

## 🌟 Features

- Conversational chatbot interface
- Sentiment analysis using Hugging Face Inference API
- Personalized suggestions based on user mood:
  - 😄 Positive → motivation tips
  - 😐 Neutral → reflective prompts
  - 😞 Negative → calming techniques and support resources
- Fully responsive UI
- Optional message log (stored locally)

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript (or React)
- **Backend :** Node.js + Express
- **API:** [Hugging Face Inference API](https://huggingface.co/inference-api)
